Web Audio API Samples
====

Web Audio APIで遊ぶ

### Description

Web Audio APIを用いていくつかの簡単なアプリケーションを実装する。

  * simple_api

    - ボタンを押すと音が鳴る仕様

    - urlを変更することで音声ファイルの取得先を変更できます

  * drum_set

    - ドラムセットもどき

    - それぞれの音を録音することも可能

    - 音はこちらから頂きました

      + [魔王魂](http://maoudamashii.jokersounds.com/)
